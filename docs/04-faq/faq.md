# FAQ

## General Questions

### Q: What browsers does TabNest support?

A: Currently, TabNest supports Chrome and Chromium-based browsers (Edge, Brave, Opera). Firefox support is coming soon.

### Q: Is TabNest free?

A: Yes, TabNest is completely free to use!

### Q: Where is my data stored?

A: Your data is stored locally in your browser using Chrome's Storage API. You can optionally sync to GitHub Gist or WebDAV.

## Features

### Q: Can I use AI features without an API key?

A: Some AI features may work with default configurations. For full features, you'll need to provide your own API keys.

### Q: What AI providers are supported?

A: TabNest supports:
- Claude (via Anthropic)
- OpenAI-compatible services
- Alibaba Cloud Bailian

### Q: How do I set up Cloud Sync?

A: See [Cloud Sync Setup](../02-features/cloud-sync) for detailed instructions.

## Troubleshooting

### Q: Why aren't my tabs saving?

A: Check that:
1. The extension has proper permissions
2. You have enough storage space
3. Your browser isn't in private/incognito mode

### Q: How do I export my data?

A: You can export your data through Settings > Data Management.
