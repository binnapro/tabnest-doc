# 常见问题

## 一般问题

### Q: TabNest 支持哪些浏览器?

A: 目前,TabNest 支持 Chrome 和基于 Chromium 的浏览器(Edge、Brave、Opera)。Firefox 支持即将推出。

### Q: TabNest 是免费的吗?

A: 是的,TabNest 完全免费使用!

### Q: 我的数据存储在哪里?

A: 你的数据使用 Chrome 的 Storage API 本地存储在浏览器中。你可以选择同步到 GitHub Gist 或 WebDAV。

## 功能

### Q: 我可以在没有 API 密钥的情况下使用 AI 功能吗?

A: 某些 AI 功能可能使用默认配置。要使用完整功能,你需要提供自己的 API 密钥。

### Q: 支持哪些 AI 提供商?

A: TabNest 支持:
- Claude(通过 Anthropic)
- OpenAI 兼容服务
- 阿里云百炼

### Q: 如何设置云端同步?

A: 查看 [云端同步设置](../02-features/cloud-sync) 以获取详细说明。

## 故障排除

### Q: 为什么我的标签页没有保存?

A: 检查:
1. 扩展具有适当的权限
2. 你有足够的存储空间
3. 你的浏览器不在隐私/无痕模式

### Q: 如何导出我的数据?

A: 你可以通过设置 > 数据管理导出数据。
