# Development Guide

See the [GitHub Repository](https://github.com/oayzz/tabnest) for complete source code and development instructions.

## Getting Started

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Run `pnpm dev` to start development
4. Load the extension in `chrome://extensions`

## Key Files

- `src/stores/appStore.ts` - Global state
- `src/components/Dashboard.tsx` - Main UI
- `src/utils/storage.ts` - Data persistence
- `src/utils/sync/` - Cloud sync implementation
