# 标签组 - 在空间内组织

标签组是空间内相关标签页的集合。它们允许你按主题、任务或任何逻辑分组进一步组织你的工作。

## 创建标签组

### 方法 1:添加分组按钮

1. 从左侧边栏选择一个空间
2. 在主区域中,点击 **"+ 添加分组"**
3. 输入分组名称(例如:"待阅读的 API"、"设计参考")
4. (可选)选择颜色和图标
5. 点击 **"创建"**

### 方法 2:从已保存的标签页创建

1. 将多个标签页保存到一个空间
2. 右键点击已保存的标签页
3. 选择 **"移动到新分组"**
4. 输入分组名称

## 了解分组功能

### 颜色编码

每个分组可以有一个颜色用于视觉组织:
- **蓝色** - API 参考、文档
- **绿色** - 活动项目、进行中
- **黄色** - 待审查、重要
- **红色** - 高优先级、紧急
- **紫色** - 学习、研究
- **灰色** - 已归档、暂停

点击分组名称旁边的颜色点即可更改。

### 图标

与空间类似,分组也可以有表情符号图标:
- 📚 文档
- 🎨 设计资源
- 🔧 工具
- 📊 分析
- 🛠️ 开发
- 📝 笔记

点击图标打开表情符号选择器。

## 管理分组中的标签页

### 添加标签页

**方法 1:保存当前标签页**
1. 在浏览器中打开标签页
2. 点击 TabNest 图标
3. 点击 **"保存当前标签"**
4. 选择目标空间和分组
5. 点击 **"保存"**

**方法 2:拖放**
1. 将 URL 或链接拖到标签组中
2. 标签页自动添加

**方法 3:右键菜单**
- 右键点击链接并选择 **"添加到 TabNest"**(如果已启用)

### 打开标签页

**打开单个标签页:**
- 点击任何标签卡片即可在浏览器中打开

**打开所有标签页:**
- 点击分组标题上的 **"打开所有"** 按钮
- 在新浏览器标签页中打开分组中的所有标签页

**在新窗口中打开:**
- Ctrl+点击(或 Mac 上的 Cmd+点击)在新窗口中打开

### 删除标签页

1. 悬停在标签卡片上
2. 点击 **"X"** 按钮
3. 确认删除

或右键点击标签页并选择 **"删除"**。

### 重新排序标签页

在分组内或分组之间拖放标签页以重新组织它们。

## 分组操作

### 重命名分组

1. 右键点击分组名称
2. 选择 **"重命名"**
3. 输入新名称
4. 按 Enter 键

或双击分组名称直接编辑。

### 折叠/展开分组

点击分组名称旁边的 **箭头图标** 以折叠或展开它。当你有很多分组并想节省空间时很有用。

### 复制分组

1. 右键点击分组
2. 选择 **"复制"**
3. 将在同一空间中创建副本

### 移动分组

1. 右键点击分组
2. 选择 **"移动"**
3. 选择目标空间
4. 点击 **"移动"**

或如果目标空间可见,使用拖放。

### 删除分组

⚠️ **警告**:此操作无法撤销!

1. 右键点击分组
2. 选择 **"删除"**
3. 确认删除

分组中的所有标签页也将被删除。

## 分组统计

悬停在分组名称上可查看:
- 分组中的标签页数量
- 上次修改时间
- 使用的总存储空间

## 技巧与最佳实践

### 命名分组

- **具体明确**:"React 组件"比"东西"好
- **适当使用动词**:"待审查"、"进行中"、"已完成"
- **保持简短**:目标是 2-4 个词
- **使用前缀排序**:"1. 研究"、"2. 设计"、"3. 开发"

### 组织模式

**按任务阶段:**
- "🔍 研究"
- "📝 规划"
- "🔨 实施"
- "✅ 测试"
- "📦 部署"

**按文档类型:**
- "📚 文档"
- "🎨 设计文件"
- "📊 数据与分析"
- "📋 需求"

**按紧急程度:**
- "🔴 关键"
- "🟠 高优先级"
- "🟡 中优先级"
- "🟢 低优先级"

### 管理分组溢出

如果空间中有很多分组:
1. **使用折叠** - 折叠不活跃的分组
2. **归档策略** - 将旧分组移动到"归档"空间
3. **定期审查** - 每月删除空的或未使用的分组
4. **子空间** - 如果太复杂,考虑创建新空间
