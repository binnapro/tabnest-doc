# 主题与外观

自定义 TabNest 的外观以匹配你的偏好,提供多种主题选项。

## 可用主题

### 浅色模式

- 简洁明亮的界面
- 最适合白天使用
- 高对比度文本
- 在明亮环境中对眼睛友好

### 深色模式

- 现代深色界面
- 减少在低光环境中的眼睛疲劳
- 适合长时间使用
- 专业外观

### 系统主题

- 自动跟随你的系统设置
- 白天浅色,晚上深色(在支持的系统上)
- 当你切换系统偏好时更改
- 与操作系统无缝集成

## 更改主题

### 方法 1:快速切换

1. 打开 TabNest 仪表盘
2. 在右上角寻找 **🌙/☀️ 主题** 按钮
3. 点击以循环切换主题

### 方法 2:设置

1. 打开 **设置**(齿轮图标)
2. 前往 **外观** 部分
3. 选择你喜欢的主题
4. 点击 **"保存"**

## 自定义选项

### 文本大小

在设置 → 外观中:
- **小** - 显示更多内容
- **中** - 默认,平衡
- **大** - 更易阅读

### 强调色

(如果在你的版本中可用)
- 选择你喜欢的强调色
- 影响按钮、链接和高亮
- 选项:蓝色、紫色、绿色、橙色、红色

### 侧边栏宽度

- 调整侧边栏宽度以更好地适应
- 拖动侧边栏边缘以调整大小
- 在小屏幕上自动折叠

## 深色模式的好处

- **减少蓝光** - 减少眼睛疲劳
- **节省电池** - 在 OLED 屏幕上
- **现代外观** - 专业外观
- **夜间更好** - 对视力更轻松

## 系统集成

### Mac (macOS)

- 尊重你的系统外观设置
- 随时间变化(如果启用)
- 与夜览集成
- 遵循你的辅助功能偏好

### Windows

- 遵循 Windows 设置 → 个性化 → 颜色
- 尊重浅色/深色模式偏好
- 随系统更改自动更新

### Linux

- 遵循 GTK 主题设置
- 尊重 GNOME 外观设置
- 可在设置中手动覆盖

## 技巧

- **提高生产力**:白天使用浅色模式,晚上使用深色模式
- **玩游戏时**:深色模式减少干扰
- **演示时**:使用你最舒适的模式
- **辅助功能**:根据需要调整文本大小
