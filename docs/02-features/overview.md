# TabNest 功能

探索让 TabNest 成为必备生产力工具的所有强大功能。

## 核心功能

### 🗂️ 空间管理
将你的标签页组织到多个独立的工作空间中,每个空间都有自己的标签组集合。

### 📦 标签组
创建和管理相关标签页的集合,支持自定义颜色和图标。

### 🤖 AI 智能分类
让 AI 使用先进的语言模型自动将你的标签页整理到有意义的分类中。

### ☁️ 云端同步
使用 GitHub Gist 或 WebDAV 在设备之间保持标签集合同步。

### 🔍  全局搜索
即时搜索所有已保存的标签页。

### 🎨 主题系统
从浅色、深色或系统偏好主题模式中选择。

### 🌍 多语言支持
提供英文和简体中文版本。
