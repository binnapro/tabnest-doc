# 基本使用

掌握 TabNest 的基础知识，开始整理你的标签页。

## 了解界面

TabNest 界面包含三个主要部分：

1. **左侧边栏** - 你的空间（工作区）
2. **主区域** - 标签组和已保存的标签页
3. **右侧面板** - 快速操作和设置

## 创建空间

空间（Spaces）是帮助你按项目或场景整理标签页的工作区。

1. 点击左侧边栏中的"**+**"按钮
2. 为你的空间输入名称（例如："工作"、"个人"、"研究"）
3. （可选）选择一个表情符号图标，便于快速识别
4. 点击"创建"

## 添加标签组

标签组是空间内相关标签页的集合。

1. 在空间内，点击"**+ 添加分组**"
2. 输入分组名称（例如："前端文档"、"API 参考"）
3. 选择一个颜色（可选）
4. 点击"创建"

## 保存标签页

### 方法 1：保存当前浏览器窗口

1. 打开你想保存的浏览器标签页
2. 点击 TabNest 图标
3. 点击"**保存当前标签**"
4. 选择目标空间和标签组
5. 点击"保存"

### 方法 2：拖放

1. 在浏览器中打开一个 URL
2. 将其拖至 TabNest 界面中的标签组
3. 标签页保存成功！

### 方法 3：右键菜单

右键点击任何链接并选择"添加到 TabNest"（如果在设置中启用）

## 打开已保存的标签页

### 打开单个标签页

点击任何已保存的标签页卡片即可在浏览器中打开。

### 打开标签组中的所有标签页

点击标签组上的"**打开所有**"按钮，一次性打开所有标签页。

## 管理标签页和标签组

### 重命名

双击空间或标签组的名称即可编辑。

### 重新排序

拖放空间和标签组以重新排列顺序。

### 删除

右键点击并选择"删除"，或使用上下文菜单。

### 折叠/展开标签组

点击标签组名称旁的箭头以折叠或展开它。
