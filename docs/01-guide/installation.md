# 安装指南

了解如何在你的浏览器上安装 TabNest。

## Chrome / 基于 Chromium 的浏览器

### 从 Chrome 网上应用店安装

1. 打开 [Chrome 网上应用店](https://chromewebstore.google.com)
2. 搜索"TabNest"
3. 点击"添加至 Chrome"
4. 确认权限对话框

### 支持的浏览器

- Google Chrome
- Microsoft Edge
- Brave
- Opera
- 其他基于 Chromium 的浏览器

## Firefox

Firefox 支持即将推出，敬请期待！

## 故障排除

### 扩展程序未出现在工具栏中

- 右键点击扩展图标并选择"固定到工具栏"
- 也可以使用快捷键 `Cmd+Shift+O`（Mac）或 `Ctrl+Shift+O`（Windows/Linux）打开 TabNest

### 权限被拒绝错误

- 确保 TabNest 拥有适当的权限
- 前往浏览器设置 → 扩展程序 → TabNest → 权限
