# Getting Started with AI Tab Categorization

TabNest's AI-powered categorization can save you hours of manual tab organization. Here's how to get started!

## Why Use AI Categorization?

- **Save Time** - Organize hundreds of tabs in seconds
- **Smart Grouping** - AI understands context and purpose
- **Easy Refinement** - Review and adjust before applying
- **Multiple Providers** - Choose the AI service you prefer

## Step 1: Set Up Your AI Provider

The easiest option is to use **Claude** from Anthropic:

1. Get an API key from [Anthropic Console](https://console.anthropic.com)
2. Open TabNest Settings → AI Configuration
3. Add your Claude API key
4. Click "Test Connection"

## Step 2: Run Categorization

1. Save tabs you want to organize
2. Click the ⚡ AI Smart Categorize button
3. Watch as AI analyzes and suggests groupings
4. Review the results
5. Click "Apply" to accept or "Cancel" to try again

## Tips for Best Results

- **Good tab titles matter** - Pages with descriptive titles categorize better
- **Start small** - Try with 10-20 tabs first
- **Iterate** - Run categorization multiple times to refine
- **Combine with manual** - AI + manual organization = perfect system

## Cost Breakdown

Using Claude:
- Most categorization: $0.05-0.15
- You pay only for what you use
- Monitor your usage in the Anthropic dashboard

## Troubleshooting

**Poor results?**
- Check that tabs have descriptive titles
- Try a smaller batch
- Try a different AI model

**Connection errors?**
- Verify your API key is correct
- Check your internet connection
- Ensure your API key has sufficient credits
