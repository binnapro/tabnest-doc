# AI 标签页分类入门指南

TabNest 的 AI 智能分类可以为你节省数小时的手动标签页整理时间。以下是入门方法!

## 为什么使用 AI 分类?

- **节省时间** - 在几秒钟内整理数百个标签页
- **智能分组** - AI 理解上下文和目的
- **轻松优化** - 在应用前审查和调整
- **多种提供商** - 选择你喜欢的 AI 服务

## 步骤 1:设置你的 AI 提供商

最简单的选项是使用 Anthropic 的 **Claude**:

1. 从 [Anthropic Console](https://console.anthropic.com) 获取 API 密钥
2. 打开 TabNest 设置 → AI 配置
3. 添加你的 Claude API 密钥
4. 点击"测试连接"

## 步骤 2:运行分类

1. 保存你想要整理的标签页
2. 点击 ⚡ AI 智能分类按钮
3. 观看 AI 分析并建议分组
4. 审查结果
5. 点击"应用"接受或"取消"重试

## 获得最佳结果的技巧

- **良好的标签页标题很重要** - 具有描述性标题的页面分类效果更好
- **从小处开始** - 先尝试 10-20 个标签页
- **迭代** - 多次运行分类以优化
- **结合手动** - AI + 手动整理 = 完美系统

## 成本分析

使用 Claude:
- 大多数分类:$0.05-0.15
- 你只需为使用的部分付费
- 在 Anthropic 仪表板中监控你的使用情况

## 故障排除

**结果不佳?**
- 检查标签页是否有描述性标题
- 尝试较小的批次
- 尝试不同的 AI 模型

**连接错误?**
- 验证你的 API 密钥是否正确
- 检查你的互联网连接
- 确保你的 API 密钥有足够的额度
