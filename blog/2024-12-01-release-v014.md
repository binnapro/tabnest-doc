# TabNest v0.1.4 发布

我们很高兴宣布 TabNest v0.1.4 发布,带来了重大改进和新功能!

## 新增内容

### ✨ 新功能

- **AI 智能分类** - 使用多个 AI 提供商(Claude、OpenAI、百炼)自动整理标签页
- **云端同步** - 将标签页同步到 GitHub Gist 或 WebDAV
- **全局搜索** - 即时查找所有空间中的任何标签页
- **深色模式** - 精美的深色主题,支持系统偏好
- **多语言** - 支持英文和简体中文

### 🐛 Bug 修复

- 修复了标签页排序问题
- 改进了大型集合的性能
- 增强了键盘导航
- 更好的错误处理

### 🚀 性能

- 加载时间提高 30%
- 减少了内存使用
- 优化了存储操作
- 更流畅的动画

## 快速开始

1. 从 Chrome 网上应用店安装 TabNest
2. 创建你的第一个空间
3. 开始保存和整理你的标签页!
